## Overview
**GIS WebView** is an evolving geospatial web interface focused on learning and applying modern GIS visualization patterns on the web.

The project explores how planetary-scale spatial data—raster imagery, vector layers, and tiled services—can be rendered efficiently in a browser using Angular and OpenLayers. The frontend handles interactive mapping and layer control, while the backend serves GIS-friendly formats such as GeoJSON, raster metadata, and map tiles.

This repository is both a learning project and a foundation for a more complete GIS visualization platform.

## Key Features
* **Unified Map Engine:** Interactive rendering of global-scale spatial data.
* **Automated Quality Control:** Integrated CI/CD using GitHub Actions for build validation and code linting.
* **Optimized Module Bundling:** Powered by Webpack for minimal load times and efficient asset management.
* **Spatial Protocol Support:** Native compatibility for GeoJSON and standard web mapping protocols.

## CI Integration
This project maintains high code standards through automated workflows:
- **Build Pipeline:** Every push to `main` triggers a webpack compilation check.
- **Super-Lint:** Enforces consistent coding styles across all supported languages.

## Project Direction
Current areas of focus include:
- Expanding planetary basemap support (Earth, Mars, Moon)
- Improving test coverage for map interactions
- Refining CI workflows for faster feedback
- Experimenting with additional spatial data formats

The goal is steady, visible improvement rather than rapid feature expansion.
